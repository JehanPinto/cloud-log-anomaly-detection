# Cloud Log Anomaly Detection

This project detects unusual behavior in cloud/system logs using Machine Learning.

## Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

## Method
- Load cloud logs (CSV)
- Preprocess data
- Train Isolation Forest model
- Detect anomalies
- Visualize results
